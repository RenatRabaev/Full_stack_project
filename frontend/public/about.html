אודות האתר

כאן מרוכזים מטרת הפרויקט, הארכיטקטורה, והסבר על כל דף — בהתאם למסמך דרישות (רשת חברתית עם React, Node, Socket.IO ומסד נתונים). השימוש בדפי HTML+CSS הוא להדגמה ויזואלית; האפליקציה המלאה רצה ב־React ומחוברת לשרת.

צד לקוח React · ממשק משתמש
צד שרת Node.js · Express
בזמן אמת Socket.IO
מסד נתונים MongoDB

מטרת הפרויקט (לפי דרישות הקורס)

לפתח מערכת חברתית בסיסית הכוללת משתמשים, פרופילים אישיים, פוסטים, ניהול בקשות חברות, ותקשורת בזמן אמת (צ'אט פרטי וקבוצתי), עם חיבור בין צד לקוח לשרת ואחסון נתונים במסד.

בצד השרת: REST API, אימות עם bcrypt ו־JWT, multer להעלאות, ו־Socket.IO לצ'אט. בצד הלקוח: ממשק רספונסיבי (Material UI), ניווט (React Router), וניהול מצב גלובלי (Redux).

מפת דפים — שימוש ודרישה

הכרטיסיות הבאות מקשרות בין קבצים (או נתיבי React) לבין מה שהמסמך דורש. קישור "תצוגה מקדימה" מוביל לאפליקציית React (מומלץ: npm run dev ב־Vite, פורט 5173). כפתור "קוד HTML" פותח חלון עם קוד המקור של קובץ ה־HTML הסטטי.

אודות (דף זה)

about.html

תיעוד מרוכז לבודקים ולמגישים: מיפוי דרישות, דפים, וטכנולוגיות. אינו חלק ממסלול המשתמש הרגיל אלא מסייע בהבנת הפרויקט.

תוצר: תיעוד (משלים ל־README).

הבהרה: שתי שכבות ממשק

  • דפי HTML+CSS בתיקייה public/ — עיצוב אחיד עם landing.css, מתאימים להצגה מהירה ולפיתוח ממשק בלי הרצת שרת (חלק מהדפים משתמשים ב־localStorage/sessionStorage לדמו מקומי).
  • אפליקציית React (Vite) — המימוש המלא מול ה־API, JWT, העלאות, וצ'אט בזמן אמת. יש להריץ backend ו־frontend לפי README.
