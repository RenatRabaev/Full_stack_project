סיכום דרישות פרויקט גמר — אפליקציית רשת חברתית
(React + Node + Socket.IO + MongoDB — מיפוי למימוש בפרויקט)

מסמך זה מסכם את מסמך הדרישות מול מה שמומש בקוד. נוצר אוטומטית לצורך הגשה / תיעוד.

חלק א׳ — סיכום לפי סעיף (מה נדרש ומה המצב)

סעיף במסמך מה נדרש מה המצב בפרויקט
מטרת הפרויקט רשת חברתית קטנה: משתמשים, פרופילים, פוסטים, בקשות חברות, צ׳אט בזמן אמת (פרטי/קבוצתי), חיבור לקוח–שרת ואחסון נתונים ממומש — אפליקציית React + API + Socket.IO + MongoDB
הרשמה / התחברות + JWT API אימות, שמירת טוקן בצד הלקוח ממומש — דפי Register/Login, POST /api/auth + JWT; הטוקן ב־sessionStorage (sn_token)
עמוד פרופיל פרטים אישיים + תמונת פרופיל ממומש — /profile ו־/users/:id
עריכת פרופיל פרטים, תמונה, ביוגרפיה ממומש — עדכון שם/ביוגרפיה/טלפון/מיקום + העלאת אווטאר
פיד פוסטים של משתמשים וחברים, לייקים ותגובות, העלאת פוסט (טקסט + תמונות) ממומש — אינטגרציה ל־/api/posts
בקשות חברות רשימת משתמשים, שליחה/אישור/דחייה, רשימת חברים ממומש — דף Friends + נתיבי /api/friends
צ׳אט פנימי (Socket.IO) רשימת שיחות (1:1 או קבוצתי), שליחה/קבלה בזמן אמת, שמירת היסטוריה ממומש — Chat.jsx + Socket.IO בשרת + Message + Conversation ב־DB
עיצוב ו־UX רספונסיבי (Tailwind / MUI), Router, Context / Redux ממומש — Material UI (MUI), React Router, Redux Toolkit
שרת — Authentication bcrypt, JWT ממומש — bcrypt + JWT
Users API CRUD, חיפוש, פרופיל ממומש ברובו — me, עדכון, אווטאר, חיפוש, צפייה לפי id, מחיקת חשבון
Friendship API שליחה/אישור/דחייה, רשימת חברים ממומש
Posts API פוסט + מדיה, לייקים, תגובות, פיד ממומש
Chat (Socket.IO) חיבור בזמן אמת, rooms, הודעות, שמירה ב־DB ממומש
File uploads multer / ענן, קישורי מדיה ב־DB ממומש — multer ל־backend/uploads
מודל נתונים User, Post, Message, Conversation, FriendRequest ממומש — תחת backend/models/
מסד נתונים MongoDB או SQL ממומש — MongoDB + Mongoose; אינדקסים בחלק מהמודלים
אופציונלי התראות, Stories, Dark mode, עדכון פיד בזמן אמת, Deployment לא ממומש (לפי README)
תוצרים Repository, README, מערכת רצה, סרטון 5–10 דקות ריפו + README + קוד רץ; סרטון — תוצר חיצוני

בנוסף: דפי HTML+CSS סטטיים ב־frontend/public/ (דף נחיתה, התחברות, הרשמה, פיד, פרופיל) — הדגמה/ללא שרת, מעבר לדרישת React במסמך הקורס.

  • Tailwind — לא בשימוש; במקום זה Material UI.
  • Context API — לא; במקום Redux Toolkit.
  • הרחבות אופציונליות + פריסה (Deployment) + סרטון — לא ממומשים בקוד (הסרטון הוא תוצר נפרד).
