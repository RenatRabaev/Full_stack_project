מדריך משתמש — דפי HTML סטטיים

תיאור קצר של כל דף תחת frontend/public/. הנתונים נשמרים מקומית (דפדפן): חשבונות ב־localStorage, מצב מחובר ופעילות סשן ב־sessionStorage.

הערה: אם אתה מפעיל את האתר דרך Vite/build, קבצים זהים עשויים להופיע גם תחת frontend/dist/.

landing.html — דף הבית

  • מציג את רעיון הפרויקט, כפתורי קריאה לפעולה (הרשמה / פיד).
  • מערכת משתמשים: קרוסלה של כרטיסי משתמשים (רשומים + דמו), עם אינדיקציה אם משתמש «מחובר».
  • שלוש כרטיסי תכונות (פרופילים / פוסטים / בקשות חברות) פועלים כלחצנים: מחובר — מעבר ישיר ליעד; לא מחובר — מעבר ל־login.html ושמירת יעד להמשך אחרי התחברות מוצלחת.
  • אזור צ'אט מתחלף (דוגמה) עם קישור לפיד.
  • כותרת: מצב אורח / שם משתמש (דרך static-user-nav.js).

login.html — התחברות

  • אימות מול רשימת החשבונות ב־localStorage.
  • אחרי התחברות מוצלחת: מעבר ליעד שנשמר מדף הבית (אם נלחצה קוביית תכונה לפני ההתחברות), אחרת ברירת מחדל ל־feed.html.
  • מודל «שכחתי סיסמה» (דמו); קישור login.html#forgot פותח את המודל.

register.html — הרשמה

  • יצירת חשבון חדש ושמירה ב־localStorage.
  • בדיקת מייל כפול; הצעה להתחברות או איפוס סיסמה.
  • אחרי הרשמה: כניסה אוטומטית (שם ומייל בסשן) ומעבר ל־feed.html.

feed.html — פיד

  • מחבר פוסטים (טקסט ותמונה אופציונלית), רשימת פוסטים סטטית + פוסטים שנשמרו בסשן.
  • לייקים ותגובות לכל פוסט; היסטוריית תגובות שלך לצורך תצוגה בפרופיל.
  • כפתורי «חבר/ה» ליד מחברי פוסטים (דרך static-friends.js).
  • קישור feed.html#post-<id> גולל לפוסט, פותח תגובות ומדגיש את הכרטיס זמנית.
  • לחיצה על תמונת פוסט פותחת תצוגה במסך מלא.

profile.html — פרופיל

  • דורש משתמש מחובר; אחרת הפניה ל־login.html.
  • טאבים: תצוגת פרופיל, גלריה, עריכת פרטים, בקשות חברות, צ'אט (דמו).
  • קישור profile.html#requests (או #tab-requests, #friends) פותח ישירות את טאב בקשות החברות.
  • פעילות אחרונה כוללת קישורים לפוסטים ותגובות מהפיד.

about.html — אודות

  • תוכן הסבר על הפרויקט ומיפוי לדרישות.
  • קישורים לתצוגה מקדימה של ממשק React (כשהשרת רץ) וכפתורים לצפייה בקוד מקור של קבצי HTML (מודל, דרך static-about-modal.js).

לפירוט פונקציות בקוד ראה תיעוד פונקציות לפי דף.
